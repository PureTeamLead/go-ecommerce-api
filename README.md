# E-Commerce Backend Service

## Features
- JWT authorization
- Endpoints to manage products and users
- Graceful shutdown
- Migrations
- Docker conterization
- Exportable logger(based on Uber's Zap)
- Monolithic architecture(modular structure)
- PostgreSQL
